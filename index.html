¿Qué aprendí en este curso?

Es un curso muy provechoso en el cual aprendí muchas cosas en tan poco tiempo. Me alegra el hecho saber que soy yo la que estoy haciendo esta página y que hace unas semanas no sabía qué era JavaScript ni mucho menos CSS.

En la primera clase aprendí a crear una página web desde cero, saber qué es un sitio web y cómo funciona, qué es UI y UX, la diferencia entre front-end y back-end; conceptos clave como accesibilidad, credibilidad, diseño web, diseño gráfico, utilidad y usabilidad...también conocí un poco sobre la arquitectura de la información, wireframes y prototipado de un sitio web.

En la segunda clase aprendí todo lo relacionado a HTML, como las etiquetas, todos sus elementos, elementos en bloque y en línea, atributos, qué es una web semántica y sus ventajas, herramientas clave que nos ayudan a leer más fácil el código y a facilitar la hora de programar, anidación y estructura básica de una página, head y body. También aprendí a usar Github y a crear repositorios, al igual que usar Git en donde pasábamos workflow de local a remoto.

En la tercera clase vimos conceptos de CSS, sus nuevas propiedades, propiedades de uso común, CSS backgrounds, relacionamos que la estructura de CSS trata mucho sobre cajas, ajustando su tamaño, color y posición; tipos de display, height y width, float, position, CSS units; estilos como CSS inline y externo, selectores, qué es la sintaxis, cascada y herencia; especificidad y herramientas clave que nos ayudan a una programación más fácil de CSS.

Y en la cuarta clase aprendí el uso de Java Script, su concepto, variables, data types, operadores, estructuras de control como if-else, if-else if, switch, loops como for y while, los objetos y sus propiedades, funciones y herramientas clave que facilitan la programación.

Mi clase favorita fue la de CSS ya que ahí aprendimos a darle color y forma (estéticamente hablando) al código, es como darle nuestro toque.
